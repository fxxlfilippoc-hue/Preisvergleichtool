Alle Strategien im Überblick Günstigster − x €/% → Zielpreis liegt x € oder x % unter dem günstigsten Konkurrenten. Gut für aggressive Preisführerschaft.
Günstigster + x €/% → Zielpreis liegt knapp über dem günstigsten Konkurrenten. Maximiert Marge bei guter Position.
UVP − Rabatt % → Fixer Rabatt auf die UVP unabhängig vom Markt. Gut als Catchall-Regel (P1).
Zielposition → Strebt Platz 1 (−10 €) oder Platz 2+ (gleicher Preis) an.
Min / Max % → Definiert einen Preiskorridor. Min Preis = niedrigster Zielpreis (z.B. 30% Rabatt = UVP×0,70). Max Preis = höchster Zielpreis (z.B. 20% Rabatt = UVP×0,80). Der Zielpreis folgt dem Markt innerhalb dieses Korridors.
